Installation
============

The preferred way to install this extension is through `composer <https://getcomposer.org/download/>`__.

Either run

``php composer.phar require --prefer-dist beastbytes/yii-leaflet``

or add

``"beastbytes/yii-leaflet": "{{versionConstraint}}"``

to the ``require`` section of your composer.json.

Leaflet Package
---------------

Install Leaflet using your chosen package manager, e.g.

``pnpm add leaflet``

``yarn add leaflet``

or add Leaflet to the dependencies of your package.json.

``"leaflet": "^1.9"``
