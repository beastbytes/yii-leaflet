Position Enum
=============

.. php:namespace:: BeastBytes\Yii\Leaflet\Controls

.. php:enum:: Position : string

    Defines the position of a control on a map

    Use as the value to the `position` key in control options

    If a position is not specified, the default position - :php:case:`Position::topRight` - is used

    .. php:case:: bottomLeft : 'bottomleft'
    .. php:case:: bottomRight : 'bottomright'
    .. php:case:: topLeft : 'topleft'
    .. php:case:: topRight : 'topright'
